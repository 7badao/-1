## 二、SDK 详细说明

### 2.1 高德地图 SDK

**服务商**：高德软件有限公司

**使用目的**：

- 为用户提供位置分享功能
- 显示地图和位置信息
- 提供周边搜索服务

**收集的信息类型**：

- 位置信息（经纬度）
- 设备信息（设备型号、操作系统版本）
- 网络信息（IP地址）

**隐私政策**：<https://lbs.amap.com/pages/privacy/>

---

### 2.2 腾讯地图 SDK

**服务商**：深圳市腾讯计算机系统有限公司

**使用目的**：

- H5 端地图展示
- 位置服务

**收集的信息类型**：

- 位置信息
- 设备信息

**隐私政策**：<https://privacy.qq.com/document/preview/dbd484ce652c486cb6d7e43ef12cefb0>

---

### 2.3 UniPush 推送服务

**服务商**：数字天堂（北京）网络技术有限公司

**使用目的**：

- 向用户推送新消息通知
- 推送系统公告

**收集的信息类型**：

- 设备标识符（ClientID）
- 设备信息（设备型号、系统版本）
- 应用信息（应用包名、版本号）

**隐私政策**：<https://www.dcloud.io/privacy.html>

---

### 2.4 华为推送 SDK

**服务商**：华为技术有限公司

**使用目的**：

- 向华为设备用户推送消息通知
- 提供稳定的消息推送通道

**调用的系统权限**：

| 权限 | 权限描述 | 使用目的 |
|-----|---------|---------|
| 获取网络状态权限 | 获取网络信息状态 | 用于和华为推送服务端交互，接收推送消息 |
| 访问网络权限 | 访问网络 | 用于访问网络，与推送服务端通信 |

**收集的信息类型**：

- 应用信息（应用包名、版本号、签名信息）
- 设备信息（设备型号、品牌、操作系统版本、系统设置、屏幕分辨率）
- 设备标识符（AAID、Push Token）
- 网络信息（网络状态、运营商信息、SSID、IP地址）

**隐私政策**：<https://developer.huawei.com/consumer/cn/doc/HMSCore-Guides/privacy-statement-0000001050042021>

---

### 2.5 DCloud 统计分析

**服务商**：数字天堂（北京）网络技术有限公司

**使用目的**：

- 分析应用使用情况
- 优化产品体验
- 排查应用崩溃问题

**收集的信息类型**：

- 设备信息（设备型号、操作系统）
- 应用使用数据（启动次数、页面访问）
- 崩溃日志

**隐私政策**：<https://www.dcloud.io/privacy.html>

---

### 2.6 KJ-VideoThumbnail 视频缩略图插件

**服务商**：DCloud 插件市场第三方开发者

**使用目的**：

- 获取本地和网络视频的缩略图、封面
- 优化视频消息列表展示体验

**收集的信息类型**：

- 视频文件路径信息

**插件地址**：<https://ext.dcloud.net.cn/plugin?id=1577>

---

### 2.7 KJ-Key 按键监听插件（仅 iOS）

**服务商**：DCloud 插件市场第三方开发者

**使用目的**：

- 监听设备音量键状态
- 检测静音开关状态

**收集的信息类型**：

- 设备音量状态信息

**插件地址**：<https://ext.dcloud.net.cn/plugin?id=11862>

## 三、系统权限说明

### 3.1 Android 权限

| 权限名称 | 权限说明 | 使用目的 |
|---------|---------|---------|
| 通知权限 | 允许应用发送通知 | 接收新消息推送通知 |
| 访问网络 | 访问网络 | 与服务器通信、推送服务 |
| 获取网络状态 | 获取网络信息状态 | 判断网络连接状态，接收推送消息 |

| 粗略位置 | 获取大致地理位置 | 位置分享、附近的人 |
| 相机 | 访问设备摄像头 | 拍照、视频通话、扫码 |
| 麦克风 | 访问设备麦克风 | 语音消息、语音通话 |
| 存储 | 读写设备存储 | 保存图片、视频、文件 |
| 电话状态 | 读取设备识别码 | 设备唯一标识、安全验证 |

| 权限名称 | 权限说明 | 使用目的 |
|---------|---------|---------|
| 相册 | 访问设备相册 | 选择图片、视频发送 |
| 相机 | 访问设备摄像头 | 拍照、视频通话、扫码 |

## 四、数据安全措施

我们采取以下措施保护您的数据安全：

1. **传输加密**：所有数据传输均采用 HTTPS 加密协议
2. **存储加密**：敏感数据在本地存储时进行加密处理
3. **访问控制**：严格的数据访问权限管理
4. **定期审计**：定期对第三方 SDK 进行安全审计

## 五、用户权利

您对个人信息享有以下权利：

1. **查询权**：您可以查询我们收集的您的个人信息
2. **更正权**：您可以要求更正不准确的个人信息
3. **删除权**：您可以要求删除您的个人信息
4. **撤回同意**：您可以随时撤回对数据收集的同意

如需行使上述权利，请通过应用内"设置-关于我们-联系客服"与我们联系。
